## Sensor Hardware

* Arduino Nano 33 IoT with Headers [ABX00032]
* HiLetgo DHT22/AM2302 | Temperature/Humidity
* Generic USB micro-B power supply

**Required Libraries**
* WifiNINA (Latest Version/Latest Firmware)
* DHTxx Sensor Library

## External Software

* MySQL Server (8.x)
* Apache2 Server (PHP & Script)
* myPHPAdmin (Optional)
